# Employee Portal Design

## 1. Overview

The Employee Portal provides self-service capabilities for employees to manage their profile, apply for leaves, view documents, and access company information.

## 2. Portal Access

### 2.1 Access Levels

| User Type | Portal Access | Capabilities |
|-----------|---------------|--------------|
| Employee | Full Employee Portal | All self-service features |
| Manager | Employee Portal + Team View | Approve leaves, view team |
| Candidate | Candidate Portal | Document upload, offer view |
| Lifetime | Lifetime Portal | Read-only document access |

### 2.2 Portal Routing

```
employee.company.com     → Employee Portal (active employees)
candidate.company.com    → Candidate Portal (pre-onboarding)
lifetime.company.com     → Lifetime Portal (ex-employees)
```

## 10. Mobile Responsiveness

The Employee Portal should be fully responsive and provide a good experience on mobile devices. Key considerations:

- Collapsible navigation menu
- Touch-friendly buttons and inputs
- Simplified views for smaller screens
- Quick access to common actions (apply leave, view balance)
- Push notifications support
